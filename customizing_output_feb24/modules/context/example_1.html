Example 1: Priority stack (what matters most)

Teach the model what to optimize for by ranking context into priority tiers.

← Home Example 2 →

Try this prompt

Copy/paste into your LLM. Keep the labels intact.

You are advising a student choosing a laptop.

GENERAL CONTEXT (nice-to-have):
- High RAM improves multitasking.
- Faster CPUs improve computation speed.
- SSDs improve file access speed.

PRIORITY CONTEXT (most important):
The student is a graphic design major using Adobe Creative Cloud daily.
Color accuracy and GPU performance are critical.
Battery life matters for studio work.

Question:
Recommend laptop specifications for this student, prioritized by impact.
Output:
1) Top 5 specs to prioritize (bullets)
2) One sentence trade-off warning

What this demonstrates

  • Context weighting via explicit priority tiers
  • Goal shaping (optimize for display/GPU rather than generic specs)
  • Structured output for comparability

Facilitator tip

Run it twice: once with priorities, once without. Compare what rises to the top.
