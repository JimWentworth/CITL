Example 5: Grounding to a provided reference (bounded knowledge)

Demonstrate document-grounded answers by restricting the model to a supplied mini-knowledge base.

← Example 4 Example 6 →

Try this prompt

Copy/paste into your LLM. Keep the labels intact.

Reference notes (use ONLY these notes):
- Photosynthesis converts light energy into chemical energy.
- It occurs in chloroplasts.
- Chlorophyll absorbs light.

Instruction:
Answer the question using only the reference notes. If the notes do not contain the answer, say: "Not in the notes."

Question:
Where does photosynthesis occur, and what absorbs light?

What this demonstrates

  • Knowledge bounding to reduce hallucination
  • Explicit refusal behavior when evidence is missing
  • RAG-style prompting without retrieval tooling

Facilitator tip

Ask a question that is *not* covered (e.g., 'What gas is produced?') to test refusal.
