Example 4: Temporal context + state (history matters)

Show that ordering and explicit state updates change interpretation and recommendations.

Try this prompt

Copy/paste into your LLM. Keep the labels intact.

You are tutoring a student. Maintain a simple state summary.

Conversation log:
Turn 1: Student struggled with algebra and avoided practice.
Turn 2: Student practiced 20 minutes daily for 12 weeks.
Turn 3: Student now solves linear equations quickly but struggles with word problems.

Task:
1) State summary (2 bullets)
2) Likely explanation for improvement (1 paragraph)
3) Next 3 tutoring steps (bullets), each tied to the state

What this demonstrates

  • History-aware reasoning
  • State summarization as a context compression tactic
  • Action selection conditioned on current state

Facilitator tip

Insert a contradictory new turn and add a rule for how state should resolve conflicts.
