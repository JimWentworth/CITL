Example 3: Persona + audience model injection

Shape explanations by injecting a learner profile (knowledge level, preferences, constraints).

← Example 2 Example 4 →

Try this prompt

Copy/paste into your LLM. Keep the labels intact.

Audience profile:
- Beginner programmer
- Overwhelmed by jargon
- Learns best via concrete examples

Instruction:
Explain recursion to this audience.

Constraints:
- Use exactly 2 analogies
- Include 1 short code example (10 lines max)
- End with 3 practice questions

What this demonstrates

  • Audience adaptation using injected user context
  • Constraint-based tailoring (analogy count, code length)
  • Teaching design embedded in context

Facilitator tip

Swap the profile (e.g., ‘math major who hates analogies’) and observe the style shift.
