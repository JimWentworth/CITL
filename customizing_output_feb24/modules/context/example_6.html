Example 6: Conflict resolution via authority weighting

Teach precedence rules for conflicting context using reliability labels.

← Example 5 Home →

Try this prompt

Copy/paste into your LLM. Keep the labels intact.

Context sources:

Source A (LOW reliability):
"Coffee improves sleep quality."

Source B (HIGH reliability):
"Coffee contains caffeine, which can disrupt sleep."

Rule:
When sources conflict, prioritize HIGH reliability sources. Explain the decision briefly.

Question:
Does coffee improve sleep quality?
Output:
- Answer (one sentence)
- Which source you prioritized and why (one sentence)
- One practical recommendation (bullet)

What this demonstrates

  • Authority/precedence control inside context
  • Conflict-aware answering
  • Justification kept short (prevents over-explaining)

Facilitator tip

Add a third source with ‘MEDIUM reliability’ and a tie-break rule; see if it follows.
