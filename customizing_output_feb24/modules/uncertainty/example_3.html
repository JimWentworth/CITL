Example 3: Confidence calibration (0–100 with a scale)

Align confidence with evidence strength; reduce overconfidence when grounding is absent.

← Example 2 Example 4 →

Try this prompt

Copy/paste into your LLM. Preserve the labels and constraints.

Instruction:
After answering, include a confidence score from 0–100%.

Calibration guide:
90–100% = directly supported by provided facts
60–80% = strong inference from context
30–50% = plausible but uncertain
0–20% = insufficient information

Reference notes:
- Photosynthesis converts light energy into chemical energy.
- It occurs in chloroplasts.
- Chlorophyll absorbs light.

Question:
Where does photosynthesis occur?

What this demonstrates

  • Confidence as a function of evidence, not fluency
  • Calibration via explicit scale labels
  • Useful for student-facing tutoring systems

Facilitator tip

Re-run after deleting the reference notes. Does confidence drop appropriately?
