Example 5: Iterative refinement loop (draft → critique → revise)

Teach multi-pass workflows that reliably improve output quality.

← Example 4 Example 6 →

Try this prompt

Copy/paste into your LLM. Preserve the step labels and constraints.

You will write an explanation of neuroplasticity for first-year undergrads.

Pass 1 (Draft):
Write ~150 words.

Pass 2 (Critique):
List 5 specific weaknesses using this rubric:
- clarity
- correctness
- completeness
- audience fit
- oversimplification risk

Pass 3 (Revise):
Rewrite the explanation, addressing each weakness.

Constraint:
In Pass 3, include a brief “limitations” sentence acknowledging uncertainty.

What this demonstrates

  • Quality gains from structured self-critique
  • Rubric-based revision is more stable than ‘make it better’
  • Built-in epistemic humility (limitations)

Facilitator tip

Have participants replace the rubric with one from their program outcomes or QM alignment.

Suggested comparison

Single-shot run

Ask once without structure; compare completeness and coherence.

Structured run

Use the scaffold; check whether reasoning is more stable and auditable.
