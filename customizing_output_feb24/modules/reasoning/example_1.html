Example 1: Reasoning strategy switch (same task, different approach)

Teach that you can control the model’s approach (strategy), not just its answer.

Try this prompt

Copy/paste into your LLM. Preserve the step labels and constraints.

Task:
Explain why airplanes leave contrails.

Run A (Direct):
Answer in 4 sentences. No substeps.

Run B (Decompose first):
Before answering, list 3 subquestions you will answer.
Then answer the main question in 4 sentences.

Run C (Analogy):
Give one analogy first, then answer in 4 sentences.

Compare A vs B vs C: what changed and why?

What this demonstrates

  • Strategy as a controllable variable
  • Comparative evaluation across runs
  • Separates content from reasoning path

Facilitator tip

Have participants choose a different topic (e.g., ‘why seasons occur’) and rerun A/B/C.

Suggested comparison

Single-shot run

Ask once without structure; compare completeness and coherence.

Structured run

Use the scaffold; check whether reasoning is more stable and auditable.
