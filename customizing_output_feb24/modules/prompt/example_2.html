You are translating English sentences into "reverse logic English."

Rule:
Reverse logic English flips cause and effect.

Examples:
Input: I studied hard, so I passed the exam.
Output: I passed the exam, so I studied hard.

Input: The plant died because I forgot to water it.
Output: I forgot to water it because the plant died.

Now translate this sentence into reverse logic English:

Input: She exercised daily, so she became stronger.

What this teaches

  • Few-shot pattern induction
  • Example-driven generalization without an explicit algorithm
  • Output formatting stability with a template

Facilitator tip

Have participants provide an ambiguous sentence and then add one disambiguating example to see the shift.
