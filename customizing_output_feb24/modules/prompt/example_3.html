Example 3: Teach a writing style (behavior shaping)

Impose a style guide and hard constraints so output becomes reliably “in-style.”

← Example 2 Example 4 →

Try this prompt

Copy/paste into your LLM. Replace bracketed placeholders as needed.

You are teaching a writing style called "compressed academic clarity."

Rules:
- Use short sentences.
- No metaphors.
- No unnecessary adjectives.
- Every sentence must communicate one clear idea.
- Maximum 12 words per sentence.

Example:

Normal writing:
Photosynthesis is the amazing process by which plants convert sunlight into energy.

Compressed academic clarity:
Photosynthesis converts sunlight into usable chemical energy.

Now rewrite this paragraph using compressed academic clarity:

Artificial intelligence can help students learn faster, but it may also reduce deep thinking if students rely on it too much.

What this teaches

  • Style transfer via explicit constraints
  • Length control at the sentence level
  • Separation of content from ornamentation

Facilitator tip

Make participants add one extra constraint (e.g., “no more than 3 sentences”) and compare outputs.
