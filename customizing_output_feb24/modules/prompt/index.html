Six ways to “teach” an LLM with prompting

How to run this as an activity

  • Have participants run an example “as-is” once, then modify one rule or anchor.
  • Compare outputs before/after; note what changed and what stayed stable.
  • Discuss failure modes: where the prompt didn’t constrain the model as intended.
