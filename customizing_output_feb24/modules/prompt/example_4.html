Try this prompt

Copy/paste into your LLM. Replace bracketed placeholders as needed.

You are a strict but fair grader. Use ONLY this rubric and the anchor examples to grade.

Rubric (0–4 each; total /12):
A) Claim clarity: 0=unclear, 2=somewhat clear, 4=precise claim
B) Evidence use: 0=none, 2=some evidence but weak connection, 4=strong evidence linked to claim
C) Reasoning: 0=illogical, 2=some logic gaps, 4=coherent reasoning with no major gaps

Anchor examples:
Example 1 (Score 10/12):
Student: "Sleep improves memory consolidation because hippocampal replay strengthens synapses. A study found better recall after sleep."
Rationale: Clear claim (4), evidence (3), reasoning (3).

Example 2 (Score 4/12):
Student: "Sleep is good. It helps you remember stuff. Everyone knows that."
Rationale: Claim (2), evidence (0), reasoning (2).

Now grade the student response below. Output EXACTLY:
- Total score: X/12
- Subscores: A=?, B=?, C=?
- 2 strengths (bullets)
- 2 improvements (bullets)
- One sentence of feedback using neutral tone

Student response:
"[PASTE STUDENT RESPONSE HERE]"

What this teaches

  • Rubric injection (explicit criteria)
  • Calibration via anchors to reduce drift
  • Constrained output format for reliability

Facilitator tip

Grade the same response with and without anchors; compare variability.
