Example 1: Teach a specific definition (concept conditioning)

Define a fictional discipline and force the model to reason inside that definition.

← Home Example 2 →

Try this prompt

Copy/paste into your LLM. Replace bracketed placeholders as needed.

You are working in a fictional discipline called Chrono-Ecology.

In Chrono-Ecology, the term "time organism" refers to any system that consumes time as a resource in order to exist.

Examples:
- A burning candle is a time organism because it consumes time as it burns.
- A human is a time organism because biological processes consume time to sustain life.
- A paused digital file is NOT a time organism because it is not actively consuming time.

Question:
Is a glacier a time organism? Explain your reasoning using the Chrono-Ecology definition.

What this teaches

  • Semantic conditioning (define terms, then enforce them)
  • Rule-following under a new ontology
  • Reasoning inside constraints (fictional but consistent)

Facilitator tip

Ask participants to add one more example that clarifies a boundary case (e.g., “a battery in storage”).
