Accessibility + AI: Practical Uses for Teaching Materials

Generative AI can support accessibility work by helping instructors create alternative formats, simplify language, draft image descriptions, and identify potential barriers in course materials. The key is to treat AI as a drafting assistant—you remain responsible for accuracy, tone, and alignment with accessibility standards.

Key idea: AI can reduce time spent on “first drafts” of accessibility work, but human review is still required—especially for technical accuracy, sensitive topics, and inclusive language.

High-Impact, Low-Effort AI Uses for Accessibility

  • Alt text drafts for instructional images (then revise)
  • Plain-language rewrites of dense instructions or policies
  • Structured headings and page organization suggestions
  • Caption and transcript cleanup (formatting, punctuation, speaker labels)
  • Accessibility “linting”: asking AI to flag likely barriers

1) Alt Text Drafting (and How to Make It Good)

Alt text should communicate what matters for the learning goal. A good workflow is:

  • Describe the instructional purpose of the image
  • Ask AI for a first draft alt text
  • Revise to include only essential details (and any data shown)
Write alt text for the image described below. INSTRUCTIONAL CONTEXT: [What is the course/topic? What should students learn from this image?] IMAGE DESCRIPTION: [What is in the image? Include any text labels, axes, or key visual elements.] REQUIREMENTS: - Keep it concise (1–2 sentences) unless the image is data-heavy. - Prioritize details needed to understand the learning point. - Do not add interpretation that is not visible in the image.
Tip: If the image is a chart, diagram, or infographic, consider adding a longer description below the image (a “long description”) with key values, trends, or steps.

Conceptual Wrap-Up

Accessibility improves learning for everyone. AI can accelerate common accessibility tasks by generating drafts—alt text, clearer language, better structure—so you can spend time on review and instructional quality rather than repetitive formatting work.

Show Expanded Explanation

AI-generated accessibility drafts are best treated as “starting points.” They can be incomplete, too vague, or overly detailed. The strongest workflows include context (what learners need to do), constraints (what not to change), and a quick human check for accuracy and tone.

Practical Guidance for Teaching and Academic Work

Pick one high-impact target: alt text for key images, clearer assignment instructions, or transcript cleanup. Use AI to draft, then revise with your course goals in mind.

If students create accessible materials, you can adapt these prompts into a learning activity: students draft alt text, compare versions, and justify which details matter for comprehension.
